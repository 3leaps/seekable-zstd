# v0.1.1 Release Notes

## Highlights

- Tag-triggered GitHub Release workflow that packages Go prebuilt libs.
- Manual minisign signing flow with checksum manifests and upload steps.
- goneat assess lint coverage for GitHub Actions (actionlint) and Makefiles (checkmake).

## Notes

- This release is Go-centric and does not publish to language registries.
- Signing remains manual; no private keys are stored in CI.
